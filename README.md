# Revolution Idle Clone

Eine Browser-basierte Incremental-Game-Prototyp-Implementierung basierend auf öffentlich zugänglichen Wiki- und Steam-Informationen zu "Revolution Idle".

## Inhalt
- `docs/DESIGN.md` — Spielstruktur, Spielmechanik und Kernkomponenten.
- `docs/PLAYTHROUGH.md` — Schritt-für-Schritt-Spielverlauf vom Start bis zum Infinity-Layer.
- `docs/SOURCES.md` — verwendete Quellen: Steam-Shop, Steam-Diskussionsseite, Wiki.
- `index.html` — Startseite für den Prototyp.
- `styles/style.css` — Oberfläche.
- `js/game.js` — Gameplay-Logik inkl. Fortschritt bis Infinity.

## Start
1. Öffne `index.html` in einem Browser.
2. Das Spiel speichert automatisch in `localStorage`.

## Hinweise
- Diese Implementierung ist eine Annäherung auf Basis der verfügbaren Wiki-Inhalte.
- Der Infinity-Layer ist implementiert und spielbar; Eternity wird freigeschaltet, sobald 1.79e308 IP erreicht sind.
